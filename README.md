## Analysis of Cardano's Incentive Mechanism

This repository hosts the "Analysis of Cardano's Incentive Mechanism" report (November 2025) and serves as the central hub for community feedback.

The goal of this report is to provide a data-driven "conversation starter" for the Cardano community to evaluate the real-world performance of our incentive design.
